# Геосетка (Geogrid)

## Концепция

Карта разбита на правильные шестиугольники (гексы). Каждый гекс — минимальная единица территории для расчётов тумана войны, знания местности и сфер влияния локаций.

## Параметры гекса

| Параметр  | Тип    | Описание                                      |
| --------- | ------ | --------------------------------------------- |
| `id`      | string | Уникальный идентификатор в формате `q:r` (см. ниже) |
| `lat`     | number | Широта центра гекса (WGS-84)                  |
| `lng`     | number | Долгота центра гекса (WGS-84)                 |

```
// Hex cell
{
  id:  string,  // "q:r" — cube/axial coordinates
  lat: number,  // center latitude
  lng: number   // center longitude
}
```

## Система координат

Используется **аксиальная система** (axial coordinates) с осями `q` (столбец) и `r` (строка). Идентификатор гекса — строка `"q:r"`, например `"14:−3"`.

Перевод аксиальных координат в географические выполняется один раз при инициализации сетки и кэшируется на сервере.

## Размер гекса

Размер подбирается так, чтобы вписаться в базовый радиус видимости игрока (200 м):

- Радиус описанной окружности гекса: **150 м**
- Расстояние между центрами соседних гексов: **~260 м**
- Один гекс полностью покрывается базовым vision_radius (200 м) с небольшим перекрытием соседей

## Соседи

Каждый гекс имеет 6 соседей. Смещения в аксиальных координатах:

```
directions = [
  (+1,  0), (+1, -1), ( 0, -1),
  (-1,  0), (-1, +1), ( 0, +1)
]
```
