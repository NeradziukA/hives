# Параметры персонажа — Человек

## Атрибуты (0–100)

Растут через опыт и экипировку.

| Атрибут         | Описание                                    | Влияет на                              |
| --------------- | ------------------------------------------- | -------------------------------------- |
| Сила            | Базовый урон в бою                          | Расчёт урона → [combat](../calculations/combat.md) |
| Защита          | Снижение входящего урона                    | Расчёт урона → [combat](../calculations/combat.md) |
| Ловкость        | Шанс попасть / уклониться                   | Вероятность попадания → [combat](../calculations/combat.md) |
| Скорость        | Перемещение, побег из боя                   | Флоу боя → [combat](../calculations/combat.md) |
| Интеллект       | Актуальность разведданных                   | LKP-задержка → [vision](../calculations/vision.md) |
| Здоровье        | Максимальный запас HP                       | Исход боя → [combat](../calculations/combat.md) |
| Лидерство       | Командный бонус к видимости группы          | Радиус видимости группы → [vision](../calculations/vision.md) |
| Зрение          | Личный радиус видимости                     | Туман войны → [vision](../calculations/vision.md) |
| Уровень вакцины | Сопротивление заражению зомби               | Вероятность заражения → [combat](../calculations/combat.md) |
| Размер рюкзака  | Количество слотов для предметов/артефактов  | Инвентарь                              |

## Навыки (0–100)

| Навык          | Описание                                                    | Влияет на                              |
| -------------- | ----------------------------------------------------------- | -------------------------------------- |
| Тяжёлое оружие | Усиливает урон тяжёлого оружия                              | Расчёт урона → [combat](../calculations/combat.md) |
| Двуручный бой  | Увеличивает урон и точность в ближнем бою                   | Расчёт урона → [combat](../calculations/combat.md) |
| Маскировка     | Снижает радиус обнаружения противником                      | Туман войны → [vision](../calculations/vision.md) |
| Регенерация    | Ускоряет восстановление HP после боя                        | Таймер восстановления → [combat](../calculations/combat.md) |

## Роли

Роль определяет специализацию и тип локации, в которой персонаж эффективен.

| Роль         | Описание                                                          | Связанная локация  |
| ------------ | ----------------------------------------------------------------- | ------------------ |
| Солдат       | Боевая специализация, усиливает Тренировочную базу                | Тренировочная база |
| Учёный       | Исследовательская специализация, усиливает Лабораторию            | Лаборатория        |
| Генерал      | Командная роль; Лидерство даёт двойной бонус к видимости группы   | База сопротивления |
| Квест мастер | Генерирует квесты союзникам в радиусе влияния                     | Любая база         |
| Босс         | Бессмертный лидер фракции                                         | Военная база       |

## Исходы боя

| Исход   | Условие                             | Последствие              |
| ------- | ----------------------------------- | ------------------------ |
| Убит    | HP ≤ 0                              | Восстановление 4 ч       |
| Заражён | Бросок заражения успешен            | Трансформация через 24 ч |
| Убежал  | Скорость позволила покинуть радиус  | Без штрафа               |
| Победил | Противник убит или убежал           | —                        |
