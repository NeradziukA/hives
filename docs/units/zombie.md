# Параметры персонажа — Зомби

## Атрибуты (0–100)

Растут через опыт и мутации.

| Атрибут         | Описание                                    | Влияет на                              |
| --------------- | ------------------------------------------- | -------------------------------------- |
| Сила            | Базовый урон в бою                          | Расчёт урона → [combat](../calculations/combat.md) |
| Защита          | Снижение входящего урона                    | Расчёт урона → [combat](../calculations/combat.md) |
| Ловкость        | Шанс попасть / уклониться                   | Вероятность попадания → [combat](../calculations/combat.md) |
| Скорость        | Перемещение, преследование жертвы           | Флоу боя → [combat](../calculations/combat.md) |
| Интеллект       | Актуальность разведданных                   | LKP-задержка → [vision](../calculations/vision.md) |
| Здоровье        | Максимальный запас HP                       | Исход боя → [combat](../calculations/combat.md) |
| Лидерство       | Командный бонус к видимости роя             | Радиус видимости роя → [vision](../calculations/vision.md) |
| Зрение          | Личный радиус видимости                     | Туман войны → [vision](../calculations/vision.md) |
| Уровень мутации | Сила заражения при атаке                    | Вероятность заражения → [combat](../calculations/combat.md) |
| Размер рюкзака  | Количество слотов для предметов/артефактов  | Инвентарь                              |

## Навыки (0–100)

| Навык         | Описание                                                     | Влияет на                              |
| ------------- | ------------------------------------------------------------ | -------------------------------------- |
| Двуручный бой | Увеличивает урон и точность в ближнем бою                    | Расчёт урона → [combat](../calculations/combat.md) |
| Зловоние      | Демаскирует людей в радиусе; нейтрализует Маскировку на 100  | Туман войны → [vision](../calculations/vision.md) |
| Регенерация   | Ускоряет восстановление HP после боя                         | Таймер восстановления → [combat](../calculations/combat.md) |

| ------------- | ----------------------------------------------------------------- | ----------------- |

| Босс          | Бессмертный лидер фракции                                         | Инкубатор         |
| Трансформация | Переходный статус: заражённый человек до смены фракции            | — (переходный)    |

## Трансформация

Особый путь появления зомби — превращение заражённого человека.

| Этап                | Условие / Таймер                                                        |
| ------------------- | ----------------------------------------------------------------------- |
| Заражение           | Бросок заражения успешен в бою с зомби                                  |
| Инкубация           | 24 часа с момента заражения                                             |
| Трансформация       | По истечении 24 ч — смена фракции                                       |
| Стартовые параметры | Наследуются от человека; Уровень мутации = 1, Уровень вакцины = 0       |

---

## Исходы боя

| Исход   | Условие                             | Последствие        |
| ------- | ----------------------------------- | ------------------ |
| Убит    | HP ≤ 0                              | Восстановление 4 ч |
| Заразил | Бросок заражения успешен            | Победа             |
| Убежал  | Скорость позволила покинуть радиус  | Без штрафа         |
| Победил | Противник убит или убежал           | —                  |
