# Расчёты — Бой

## Вероятность попадания

```
PD = agility_attacker / agility_defender * 100%
```

| Параметр           | Источник                   |
| ------------------ | -------------------------- |
| agility_attacker   | Ловкость атакующего        |
| agility_defender   | Ловкость защищающегося     |

---

## Расчёт урона

```
damage = max(0, strength_attacker - defense_defender)
```

| Параметр          | Источник                   |
| ----------------- | -------------------------- |
| strength_attacker | Сила атакующего            |
| defense_defender  | Защита защищающегося       |

Навыки **Тяжёлое оружие** и **Двуручный бой** добавляют бонус к `strength_attacker` — величина бонуса TBD.

---

## Вероятность заражения

Применяется только если атакующий — зомби.

```
PI = mutation_zombie / vaccine_human * 100%
```

| Параметр      | Источник                         |
| ------------- | -------------------------------- |
| mutation_zombie | Уровень мутации зомби-атакующего |
| vaccine_human   | Уровень вакцины человека-жертвы  |

---

## Восстановление после боя

| Исход   | Базовый таймер | Влияние Регенерации       |
| ------- | -------------- | ------------------------- |
| Убит    | 4 ч            | Регенерация сокращает TBD |
| Заражён | 24 ч до трансформации | — (Регенерация не влияет) |

---

## Флоу боя

1. Атакующий входит в радиус 100 м → запрос серверу
2. Защищающийся получает уведомление, 5 мин на реакцию
3. Варианты: **Принять** / **Убежать** (Скорость) / **Звать на помощь** (1 мин)
4. По истечении 5 мин — расчёт результата

Побег успешен если `speed_defender > speed_attacker` (точная формула TBD).
