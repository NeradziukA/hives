# Расчёты — Видимость

## Личный радиус видимости

```
vision_radius = 200 + vision_attribute * 10  // meters
```

| vision_attribute | vision_radius |
| ---------------- | ------------- |
| 0                | 200 м         |
| 50               | 700 м         |
| 100              | 1200 м        |

---

## Маскировка (навык людей)

Снижает эффективный радиус, в котором противник может обнаружить носителя.

```
detection_radius = vision_radius_enemy * (1 - camouflage / 100)
```

| camouflage | Эффект                                    |
| ---------- | ----------------------------------------- |
| 0          | Обнаружение в полном радиусе противника   |
| 50         | Радиус обнаружения уменьшен вдвое         |
| 100        | Полная невидимость (detection_radius = 0) |

---

## Зловоние (навык зомби)

Увеличивает эффективный радиус обнаружения людей носителем.

```
reveal_radius = vision_radius * (1 + stench / 100)
```

При расчёте видимости людей используется `reveal_radius` вместо `vision_radius`.
Зловоние = 100 нейтрализует Маскировку любого уровня.

| stench | reveal_radius           |
| ------ | ----------------------- |
| 0      | vision_radius × 1.0     |
| 50     | vision_radius × 1.5     |
| 100    | vision_radius × 2.0     |

---

## Задержка разведданных (Интеллект)

Определяет, насколько актуален LKP-снимок противника.

```
intel_delay = max(0, 60 - intelligence * 0.6)  // seconds
```

| intelligence | intel_delay |
| ------------ | ----------- |
| 0            | 60 с        |
| 50           | 30 с        |
| 100          | 0 с (реальное время) |

---

## Видимость группы (Лидерство)

Сумма атрибута Лидерство всех членов группы даёт бонус к общему радиусу видимости.
Генерал (люди) / Альфа (зомби) — Лидерство считается с коэффициентом × 2.

| Тип локации                     | Радиус раскрытия |

| Лаборатория / Мутатор           | 800 м            |
