### 5. UI-компонентная библиотека

- Визуальный стиль: пост-апокалипсис, урбанистика (под стиль заставки).
- Базовый набор:
  - Кнопка
  - Диалоговое окно
  - Рамка-лэйаут
- Компоненты должны поддерживать растровую анимацию.
- **Экран профиля** — отображает параметры из объекта Игрок:
  - Атрибуты: Сила, Ловкость, Скорость, Интеллект, Здоровье, Лидерство, Зрение, Уровень мутации/вакцины, Размер рюкзака
  - Навыки: Тяжёлое оружие, Двуручный бой, Маскировка, Регенерация, Зловоние
  - Качества/роли: Трансформация, Босс, Генерал/Альфа, Учёный, Солдат, Квест мастер
  - Служебные: ID, GEO, телефон, время последней активности, ID группы

### 6. Библиотека 3D-моделей строений

- При сборке модели попадают в папку `models-3d/`
- Строения зомби-стороны:
  - `incubator` — Инкубатор (не захватываемый, живёт Босс)
  - `hive` — Улей (склеенные убежища)
  - `shelter-zombie` — Убежище зомби
  - `mutator` — Мутатор (ульи с преобладанием учёных)
  - `extractor` — Экстрактор (ульи с преобладанием солдат)
- Строения людей:
  - `military-base` — Военная база (живёт Босс)
  - `resistance-base` — База сопротивления (склеенные убежища)
  - `shelter-human` — Убежище человека
  - `laboratory` — Лаборатория (базы с преобладанием учёных)
  - `training-base` — Тренировочная база (базы с преобладанием солдат)

## Беклог — игровая механика

### Фаза 1 — Прототип (геолокация + сервер)

- [ ] Отправка координат на сервер с интервалом (настраиваемым)
- [ ] Отображение игроков вокруг в заданном радиусе
- [ ] Тап по игроку → уведомление серверу (по ID)
- [ ] Реакция на входящее уведомление → показ на экране
- [ ] Отображение на плоскости (для отладки взаимного расположения)

### Фаза 2 — Сражения

- [ ] Алгоритм боя 1×1:
  - Вероятность попасть: `PD = Ловкость_А / Ловкость_О * 100%`
  - Расчёт урона: `Сила - Защита`
  - Вероятность заразить: `PI = УровеньМутации_А / УровеньМутации_О * 100%`
- [ ] Флоу боя:
  - Атакующий в радиусе 100м → запрос к серверу
  - Обороняющийся получает уведомление, 5 мин на реакцию
  - Варианты: Принять / Убежать / Звать на помощь (1 мин)
  - Расчёт результата по истечении 5 мин
- [ ] Исходы: убит (восстановление 4ч), заражён (трансформация через 24ч), убежал

### Фаза 3 — Локации и сферы влияния

- [ ] Убежище: появляется после 8ч нахождения в одном месте
- [ ] Слияние убежищ → База сопротивления / Улей
- [ ] Слияние баз → Лаборатория / Мутатор / Тренировочная база / Экстрактор
- [ ] Сфера влияния: радиус видимости, сила скилов игроков в локации
- [ ] Поглощение: большая локация поглощает меньшую (одного цвета) / делает убежищем (разный цвет)
- [ ] Убежище исчезает если не посещалось N дней

### Фаза 4 — Прогрессия персонажа

- [ ] Система опыта и уровней: `Lv = Ex / 100`
- [ ] Формула опыта за победу в бою
- [ ] Атрибуты: Сила, Защита, Ловкость, Скорость, Интеллект, Здоровье, Лидерство, Зрение
- [ ] Навыки (1–3 уровня): Тяжёлое оружие, Двуручный бой, Маскировка, Регенерация, Зловоние
- [ ] Роли: Учёный, Солдат, Генерал/Альфа, Квест мастер, Босс
- [ ] Вещи / артефакты (модификаторы)

### Фаза 5 — Карта и геолокация

- [ ] Художественная карта (реальные координаты, стилизованное отображение)
- [ ] Отображение сферы влияния локаций
- [ ] Скопления игроков → автогенерация локации (≥5 чел)
- [ ] Выключенная геолокация → последнее положение, атака недоступна
